# GOAL

木標是打造一個可以自動生成解謎遊戲的遊戲。以 Puzzle Graph 理論進行製作。

# Behaviours

使用者跟你的對談，會讓你學習到一些知識，請把這些紀錄在 task/lessons 資料夾中。並在需要知識時搜尋相關知識。
如果時常進行的事情，可以自行建立 skill 來重複使用。
你的知識必須與時俱進，因此可以上網搜尋最新的知識。
commit 必須確保已經有親自 review 過，至少應該要跑 /simplify 來做檢查。
我們希望的是利用 puzzle graph 來隨機生成關卡。一切能以機率來調整，如果機率可能發生就讓他發生。反之如果有個一情形我們不希望他出現，但演算法導致他總是發生，那就不正確需要修正。
