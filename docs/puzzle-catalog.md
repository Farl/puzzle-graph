# 解謎遊戲謎題目錄

蒐集自真實密室逃脫、冒險遊戲、解謎遊戲的常見謎題類型，作為模板設計參考。

---

## 一、物理操作類 (physical)

需要特定工具或鑰匙才能操作的物理機關。最經典的 lock-and-key 模式。

| 謎題 | 鑰匙 | 場景描述 |
|---|---|---|
| 掛鎖寶箱 | 鑰匙 | 找到對應鑰匙開鎖 |
| 鐵鍊鎖門 | 破壞剪 | 剪斷鎖鏈打開通道 |
| 釘死的板子 | 撬棍 | 撬開釘死的蓋板/地板 |
| 斷把手的門 | 門把零件 | 裝回門把才能開門 |
| 卡住的閥門 | 扳手 | 用扳手轉動生鏽閥門 |
| 上鎖的抽屜 | 迷你鑰匙 | 精緻小鎖配精緻小鑰匙 |
| 保險箱 | 鑰匙 | 旋轉鑰匙打開保險箱 |
| 掛鎖鐵門 | 鑰匙 | 粗重掛鎖配大鑰匙 |
| 玻璃展示櫃 | 小榔頭 | 敲碎玻璃取出物品 |
| 繩結障礙 | 剪刀/小刀 | 剪斷繩子釋放物品 |

## 二、隱藏/環境類 (hidden)

需要特定工具改變環境條件，才能發現或取得隱藏的物品。

| 謎題 | 鑰匙 | 場景描述 |
|---|---|---|
| 黑暗角落 | 手電筒 | 照亮暗處發現隱藏物品 |
| 油污覆蓋 | 濕布 | 擦拭髒污露出底下的線索 |
| 隱形文字 | UV 燈 | 紫外線照射顯示隱藏墨水訊息 |
| 磁性暗格 | 磁鐵 | 用磁鐵吸出藏在牆內的金屬物品 |
| 高處物品 | 鉤子/伸縮桿 | 勾下放在高處的東西 |
| 水底鑰匙 | 撈網 | 從水池中撈出沉底的鑰匙 |
| 結冰的鎖 | 吹風機/熱源 | 融化冰塊取出鑰匙 |
| 沙中埋藏 | 鏟子/篩子 | 從沙堆中篩出小零件 |
| 暗門書架 | 特定書本 | 抽出正確的書觸發書架暗門 |
| 畫後暗格 | 畫框移除 | 移開牆上的畫發現背後的保險箱 |
| 地毯下方 | 掀起地毯 | 地毯下有地板暗門或線索 |
| 鏡子反射 | 鏡子 | 用鏡子看到反寫的文字/反射光線 |
| 夾層空間 | 螺絲起子 | 拆開家具發現夾層裡的物品 |

## 三、密碼/線索類 (password)

需要找到密碼線索（紙條、刻字、環境線索）才能輸入。

| 謎題 | 線索來源 | 場景描述 |
|---|---|---|
| 數字密碼鎖 | 密碼紙條 | 從環境中找到 3-6 位數字 |
| 文字密碼鎖 | 密碼紙條 | 字母或詞語密碼 |
| 方向鎖 | 方向線索 | 上下左右的方向組合 |
| 日期鎖 | 日記/照片 | 從故事線索推理出重要日期 |
| 數學鎖 | 算式線索 | 環境中的數字計算出答案 |
| 顏色序列鎖 | 顏色排列線索 | 按照特定顏色順序輸入 |
| 摩斯密碼鎖 | 摩斯電碼表 | 解讀音訊或燈光的摩斯密碼 |
| 凱薩密碼鎖 | 密碼盤 | 字母位移解碼 |
| 符號替換鎖 | 對照表 | 看符號對照表翻譯出密碼 |
| 音樂密碼鎖 | 樂譜 | 按照樂譜彈奏/輸入音符 |

## 四、組合類 (combination) — 2 鑰匙

需要收集兩個物品才能解開。

| 謎題 | 鑰匙 A + 鑰匙 B | 場景描述 |
|---|---|---|
| 電子設備 | 電源線 + 隨身碟 | 通電後讀取解密資料 |
| 化學反應 | 酸性試劑 + 鹼性試劑 | 混合產生化學反應溶解封印 |
| 齒輪機關 | 大齒輪 + 小齒輪 | 安裝齒輪讓機械運轉 |
| 雙閥門 | 左閥門 + 右閥門 | 同時轉動兩個閥門開門 |
| 焊接修復 | 焊槍 + 焊條 | 焊接斷裂的金屬部件 |
| 無線電 | 天線 + 電池組 | 組裝無線電接收求救訊號 |
| 光學裝置 | 透鏡 + 稜鏡 | 折射光線照射到感應器 |
| 液壓裝置 | 液壓管 + 液壓油瓶 | 填充液壓系統啟動升降台 |
| 保險絲盒 | 保險絲 + 開關把手 | 換上保險絲再拉閘送電 |
| 天平機關 | 砝碼 A + 砝碼 B | 兩邊放上正確重量觸發機關 |
| 印章蓋印 | 印章 + 印泥 | 蓋上正確印章開啟認證鎖 |
| 訊號接收 | 頻率卡 + 接收器 | 設定正確頻率接收密碼 |
| 磁卡門 | 磁卡 + 讀卡機電池 | 讀卡機沒電，裝電池後刷卡 |

| 火箭發射台 | 燃料罐 + 點火器 + 導航晶片 | 三個零件就位才能發射 |

| 望遠鏡 | 鏡筒 + 目鏡 + 三腳架 | 組裝望遠鏡看到遠處的線索 |

| 急救箱 | 繃帶 + 消毒液 + 止痛藥 + 注射器 | 集齊急救物資打開密封門 |
| 煉金爐 | 礦石 + 催化劑 + 燃料 + 模具 | 煉製出開啟最終門的鑰匙 |

## 七、小遊戲類 (minigame)

收齊零件後觸發互動式操作，需要技巧而非單純收集。

| 謎題 | 零件 | 互動方式 |
|---|---|---|
| 管道拼圖 | 管道零件 ×2 | 旋轉管道連通入口到出口 |
| 接線盤 | 電線捲 ×2 | 左右端子配色接線 |
| 拼圖碎片 | 碎片 ×3-6 | 拼出完整圖案/地圖 |
| 滑塊謎題 | 滑塊 ×2 | 移動方塊讓目標滑出 |
| 保險箱轉盤 | 轉盤零件 ×2 | 按正確順序轉動多個轉盤 |
| 密碼盤解碼 | 密碼盤零件 ×2 | 對齊多層轉盤解碼 |
| 迷宮導航 | 鋼珠 + 操縱桿 | 傾斜迷宮讓鋼珠到達終點 |
| 記憶配對 | 卡片 ×4 | 翻牌配對記憶遊戲 |
| 音序重現 | 音叉 ×3 | 重現聽到的音符順序 |
| 鏡面反射 | 鏡片 ×3 | 調整鏡面角度讓光線到達目標 |

## 八、狀態轉換類 (state/crafting)

物品需要經過狀態轉換才能使用。通常涉及固定裝置（不可搬動的場景互動對象）。

| 產品 | 原料鎖 | 所需材料 | 場景描述 |
|---|---|---|---|
| 手電筒 | 沒電的手電筒 | 電池 ×2 | 裝電池恢復供電 |
| 撬棍 | 斷裂的撬棍 | 金屬桿 | 焊接修復 |
| 門禁磁卡 | 空白磁卡 | IC 晶片 | 寫入權限資料 |
| 破壞剪 | 鈍掉的破壞剪 | 磨刀石 | 磨利刀刃 |
| 濕布 | 乾布 | 水盆（固定） | 浸水擰乾 |
| 繩索 | 散落的纖維 | 紡錘（固定） | 搓繩 |
| 鑰匙模具 | 黏土模型 | 熔爐（固定）| 翻模鑄造 |
| 通電設備 | 斷電裝置 | 發電機（固定） | 接通電源 |
| 加熱液體 | 冰凍藥瓶 | 火源（固定） | 解凍藥劑 |
| 淨水 | 污水瓶 | 淨水器（固定） | 過濾雜質 |

## 九、觀察/搜索類 (observation)

不需要工具，靠仔細觀察環境就能找到線索或物品。

| 謎題 | 場景描述 |
|---|---|
| 數物品 | 房間裡特定類型的物品數量就是密碼的一位 |
| 找差異 | 兩幅畫/兩個場景的差異指向答案 |
| 讀標記 | 家具上的划痕、牆上的塗鴉是線索 |
| 書架排列 | 書脊上的數字/字母按順序排列得出密碼 |
| 地圖標記 | 地圖上的標記點對應房間中的位置 |
| 日記線索 | 翻閱日記找到關鍵日期或名字 |
| 照片中的線索 | 照片中的物品排列暗示密碼 |
| 鐘面時間 | 牆上時鐘停在的時間就是密碼 |
| 顏色排列 | 房間中物品的顏色順序對應鎖的顏色序列 |
| 影子投射 | 物品在光線下投射出的影子形成文字或圖案 |

## 十、邏輯/推理類 (logic)

需要運用邏輯推理、數學計算或模式識別。

| 謎題 | 場景描述 |
|---|---|
| 數學算式 | 牆上的數學題，答案是密碼 |
| 數列規律 | 給出一串數字，找出規律推算下一個 |
| 邏輯格 | 根據多條線索推理出正確的組合 |
| 符號邏輯 | 每個符號代表一個數字，列出等式求解 |
| 路徑推理 | 從迷宮中找出正確路徑，路徑上的字母組成密碼 |
| 排列組合 | 按照特定規則排列物品的順序 |
| 真假判斷 | 多個陳述中找出真的那個，指向答案 |
| 地圖導航 | 按照指令在地圖上移動，最終位置是答案 |
| 倒推法 | 給出結果和過程，倒推起始狀態 |

## 經典遊戲謎題參考

從知名解謎遊戲中提取的經典謎題機制，按來源遊戲分類。

### The Room 系列
精密機械箱體操作，層層拆解。

| 謎題 | 機制 | 場景描述 |
|---|---|---|
| 多層機關箱 | 依序解開外層鎖 → 露出內層 | 打開一層發現下一層有新機關 |
| 特殊透鏡觀察 | 裝上目鏡 → 看到隱藏符號 | 用特殊鏡片看到肉眼不可見的標記 |
| 滑軌解鎖 | 沿軌道推動滑塊 → 釋放卡榫 | 把滑塊推到正確位置解除鎖定 |
| 旋轉圓盤對齊 | 轉動多個同心圓盤 → 圖案對齊 | 轉動圓環讓符號或管道連通 |
| 微型齒輪機構 | 安裝缺失齒輪 → 驅動機械 | 小型精密機械需要正確齒輪才能運作 |
| 隱藏按鈕 | 觸摸特定位置 → 觸發暗格 | 表面看不出來的凹槽或按鈕 |

### Myst 系列
環境觀察型謎題，從場景中蒐集資訊。

| 謎題 | 機制 | 場景描述 |
|---|---|---|
| 星座對應 | 觀察星圖 → 輸入對應符號 | 把看到的星座對照到操作面板 |
| 音序記憶 | 聽音樂 → 重現音符順序 | 記住旋律後在鍵盤上彈奏 |
| 塔台旋轉定位 | 旋轉塔台 → 對準特定方向 | 旋轉到正確角度揭示隱藏資訊 |
| 電壓/數值調節 | 調整數值到精確目標 | 用按鈕組合設定精確的電壓值 |
| 時鐘對時 | 從線索得知時間 → 設定時鐘 | 把時針分針撥到特定時刻 |
| 書頁線索 | 翻閱書籍 → 找到地圖或密碼 | 書中散落的線索指向解答 |
| 機械連動 | 拉一個拉桿 → 影響遠處機關 | 操作裝置後需到另一處觀察結果 |

### Resident Evil 系列
收集散落的物件嵌入機關。

| 謎題 | 機制 | 場景描述 |
|---|---|---|
| 紋章嵌入 | 收集 N 個紋章 → 放入凹槽 | 太陽/月亮/星星紋章各歸其位開啟石門 |
| 棋子插座 | 把棋子形道具插入對應孔位 | 國王、騎士等形狀的插頭對應插座 |
| 曲柄裝置 | 用曲柄啟動機械 | 曲柄轉動排水/升降/開閘 |
| 畫像謎題 | 調整畫像角度或順序 | 旋轉肖像畫讓目光指向同一處 |
| 藥草混合 | 收集不同藥草 → 混合 | 紅藍綠藥草組合出不同效果 |
| 石板拼圖 | 收集碎裂石板 → 拼合 | 拼合後放入牆面凹槽開門 |

### Zelda 系列
動作解謎結合，需要工具改變環境。

| 謎題 | 機制 | 場景描述 |
|---|---|---|
| 推箱踩板 | 推方塊到壓力板上 | 壓下開關打開門或降下橋 |
| 點火順序 | 用火把依序點燃火台 | 限時內點完所有火台開門 |
| 鏡面反光 | 調整鏡面角度反射光線 | 光線路徑到達感應點觸發機關 |
| 水位控制 | 切換水位高低 | 改變水位讓通道浮出或沉入 |
| 水晶開關 | 攻擊水晶 → 切換狀態 | 紅/藍方塊交替升降 |
| 鉤爪到達 | 用鉤爪飛到對面 → 拉動開關 | 需要特定工具才能到達的操作點 |
| 磁力操控 | 用磁力移動金屬物體 | 遠距操控金屬平台或鑰匙 |

### The Witness
路徑規則型謎題，從環境學習規則。

| 謎題 | 機制 | 場景描述 |
|---|---|---|
| 線條迷宮 | 從起點畫線到終點 | 路徑必須通過所有標記點 |
| 對稱路徑 | 同時控制兩條對稱的線 | 一條線向上另一條向下 |
| 顏色分區 | 畫線把不同顏色區隔開 | 同色方塊必須在同一區域 |
| 形狀填充 | 路徑圍出的區域需包含指定形狀 | 俄羅斯方塊形狀必須剛好放進去 |
| 環境投影 | 環境中的影子/樹枝就是路徑 | 從正確角度看環境，線索映射到面板上 |

### Rusty Lake / Cube Escape 系列
超現實風格，物品組合和煉金術。

| 謎題 | 機制 | 場景描述 |
|---|---|---|
| 煉金合成 | 組合基礎元素 → 產出新物質 | 在煉金爐中混合材料 |
| 時鐘撥盤 | 設定正確時間 → 觸發事件 | 鐘面時間對應環境線索 |
| 符號序列輸入 | 記住符號順序 → 依序輸入 | 從場景各處收集符號後輸入面板 |
| 滑動拼圖 | 滑動方塊復原圖案 | 經典 15 拼圖 |
| 儀式步驟 | 按正確順序執行多個動作 | 先放道具、再點蠟燭、最後唸咒語 |
| 多房間聯動 | A 房間的操作影響 B 房間 | 需在不同場景之間來回收集線索 |

### Machinarium
無文字冒險，純粹靠觀察和物品操作。

| 謎題 | 機制 | 場景描述 |
|---|---|---|
| 機械修復 | 收集零件 → 修復故障機器 | 找到缺失零件讓機器恢復運作 |
| 環境互動鏈 | A 觸發 B → B 解鎖 C | 一連串的因果互動 |
| 思維泡泡線索 | NPC 的想像畫面就是提示 | 觀察角色想什麼來推理需求 |
| 音樂演奏 | 記住旋律 → 重現 | 聽過一段音樂後在樂器上重現 |

### Professor Layton 系列
獨立邏輯謎題集。

| 謎題 | 機制 | 場景描述 |
|---|---|---|
| 滑塊拼圖 | 移動方塊讓目標通過 | 華容道類型 |
| 火柴棒重排 | 移動/移除火柴棒 → 改變等式或形狀 | 移動 N 根火柴讓算式成立 |
| 天平稱重 | 用最少次數找出不同重量的物品 | 經典邏輯推理 |
| 一筆畫 | 不重複地走過所有路徑 | 尤拉路徑問題 |
| 硬幣翻轉 | 按規則翻轉硬幣 → 達到目標狀態 | 翻一個會連帶翻相鄰的 |
| 切割拼圖 | 把圖形切成指定份數和形狀 | 只能切 N 刀分成相同的塊 |

### The Talos Principle / Portal
空間操控和光線/能量引導。

| 謎題 | 機制 | 場景描述 |
|---|---|---|
| 雷射折射 | 用稜鏡把雷射導向接收器 | 擺放折射器讓光線到達目標 |
| 傳送門 | 創造入口和出口 → 繞過障礙 | 利用空間扭曲到達不可能的位置 |
| 干擾器 | 用裝置暫時停用障礙 | 干擾力場讓路通過，但拿走就恢復 |
| 複製器 | 複製自己的行動 → 兩人合作 | 錄製一段操作，然後和錄像同時行動 |
| 連接器 | 同時連接多個能量點 | 站在正確位置讓能量線穿過所有節點 |

## 設計原則

### 複雜度層級
- **1 鑰匙**：簡單直覺，適合開場熱身
- **2 鑰匙**：中等收集，玩家需要同時記住兩個目標
- **3 鑰匙**：需要大範圍探索，適合中期核心謎題
- **4+ 鑰匙**：關卡最終機關，通常需要解完多個前置謎題

### 物品屬性
- **消耗型** (key)：使用一次後消失
- **可重複使用** (tool)：手電筒、撬棍等可重複使用的工具
- **固定裝置** (stationary)：水盆、熔爐等不可搬動的場景互動對象

### 多樣性
- 同一類別可有多個 **variation**（不同名稱和描述，同一機制）
- 模板使用 shuffle 牌堆：抽過的模板不會立刻再出現
- 同一關卡中應混合不同類別的謎題
